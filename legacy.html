Legacy OpenAPI.Tools

This is a community-driven list of high-quality OpenAPI tools that have fallen out of recency for various reasons, but are still helpful for those stuck with older OpenAPI. An open source project from APIs You Won't Hate.

⚠️ This is the legacy list of old OpenAPI tools, covering v3.0 and v2.0 tools that have not yet added support for v3.1 or v3.2. Please visit the main site for the latest tools.
